Flespi gateway
==============

Release v\ |version|. (:ref:`Installation <install>`)


**Flespi gateway** is a python library for working with telematics data from Flespi platform.

-------------------


**Flespi gateway** allows you to get data from flespi platform by sending HTTP/1.1 requests.
